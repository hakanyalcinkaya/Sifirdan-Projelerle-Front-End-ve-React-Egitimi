# React Router Dom ile Calismak

## Projenin Amaci:
React Router Dom Genel Yapisini Anlamak ve Privite Route ile Calismayi Ogrenmek

## Proje Kurulumu
```
npm i
```

## Projeyi Calistirmak Icin:
```
npm run dev
```

## Projeyi Browser'da Acmak Icin:
```
http://localhost:3000
```
